ChatApp - End-to-End Encrypted Messenger

Flutter, Node.js, E2E Encryption | 2025

Tech Stack: Flutter, Node.js, X25519, Ed25519, AES-GCM

Project Overview

A secure messaging application focused on privacy and military-grade encryption, comparable to industry standards like Signal and WhatsApp.

Key Features

  • Implemented hybrid encryption combining Diffie-Hellman key exchange (X25519) with RSA-style authentication (Ed25519) for secure session establishment.
  • Built Double Ratchet messaging with AES-GCM encryption, achieving perfect forward secrecy and post-compromise security.
