SMI - Behavior Management Platform

React Native, Node.js, PostgreSQL | 2025

Tech Stack: React Native (Expo), Node.js, PostgreSQL, UUID

Project Overview

A comprehensive cross-platform educational management system serving multiple user roles with real-time data synchronization.

Key Features

  • Built using React Native and Node.js to serve students, coaches, and admins.
  • Designed and implemented PostgreSQL database with 12+ interconnected tables using UUID primary keys.
  • Integrated speech-to-text capabilities and developed gamified learning features with points system and progress tracking analytics.
